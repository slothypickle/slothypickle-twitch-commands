SlothyPickle Commands

Pickles, chat games, stats, raffles, and stream tools.

Popular Commands

!pickles
Aliases: none
Check your current pickle balance.
Example: !pickles
!watchstats
Aliases: none
Shows watch time and pickles earned from watching.
Example: !watchstats @username
!leaderboard
Aliases: none
Shows the pickle leaderboard.
Example: !leaderboard
!optout
Aliases: none
Opt out of pickles, chat games, and related leaderboards.
Example: !optout

Chat Games

!slots
Aliases: !s, ! s, slots, s
Spin the pickle slot machine.
Example: !s 10k
!roulette
Aliases: !r, r
Bet pickles on roulette.
Example: !r red 5k
!blackjack
Aliases: !bj, bj
Start a blackjack hand.
Example: !bj 10k
!hit / !stand
Aliases: hit, h, stand, s
Use during an active blackjack hand.
Example: !hit
!duel
Aliases: none
Challenge another viewer to a pickle duel.
Example: !duel @username 10k

Raffle

!join
Aliases: none
Join an active raffle when entries are open.
Example: !join
!raffle
Broadcaster only
Starts a raffle.
Example: !raffle
!roll
Broadcaster only
Forces the raffle roll.
Example: !roll

Stats & Reports

!stats
Aliases: none
Shows your chat game stats.
Example: !stats @username
!allstats
Aliases: none
Shows broader chat game stats.
Example: !allstats
!brinereport
Aliases: none
Shows pickle economy inflation and circulation info.
Example: !brinereport
!lastseen
Aliases: none
Shows when someone was last seen in chat.
Example: !lastseen @username
